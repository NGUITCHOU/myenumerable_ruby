# 📖 myenumerable_ruby<a name="about-project"></a>

This is the first technical Codegisoft Academy project it tests more on Git, GitHub, GitHub Workflow.
It is also a project that boost your IQ, since it is more of logic (thinking)than coding

[myenumerable_ruby] is a ruby implementation that requires decoding Morse code. The code include methods to decode individual characters, words and entire messages, while adhering for the DRY principle.
